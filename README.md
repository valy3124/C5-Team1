## Quick Start

`setup.sh` automates the installation of Miniconda, creates the environment, and installs all necessary libraries dependencies:

```bash
./setup.sh
```

Then, to work within the environment:

```bash
conda activate c5
```

Finally, if you need to update dependencies, just add new packages to `requirements.txt` and run:

```bash
pip install -r requirements.txt
```
